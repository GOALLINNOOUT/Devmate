# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of DevMate.

## Reporting A Vulnerability

Please report vulnerabilities privately by opening a GitHub security advisory at:

https://github.com/GOALLINNOOUT/Devmate/security/advisories/new

If GitHub advisories are unavailable, open a minimal public issue asking for a private contact path. Do not include exploit details, secrets, or private tokens in public issues.

DevMate is a local developer CLI. It should not require telemetry, cloud credentials, or elevated privileges for normal diagnostic commands.
