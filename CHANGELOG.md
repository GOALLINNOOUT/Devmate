# Changelog

All notable changes to DevMate will be documented here.

## 0.1.5 - Unreleased

- Added project analysis, JSON tools, env inspection, Git summaries, file audits, JWT helpers, system status, doctor checks, and safe process cleanup.
- Added Windows-friendly ASCII table output.
- Added gitignore-aware project walking.
- Added `devmate setup` as a first-run onboarding command.
- Added `devmate update` with support for winget, cargo, and manual GitHub Release installs.
- Added `devmate uninstall` with confirmation, package-manager removal, and manual executable cleanup.
- Added Windows, Linux, macOS Intel, and macOS Apple Silicon release packaging.
- Added winget packaging docs and manifests for `ADELA.Devmate`.
- Expanded install documentation for GitHub Releases, winget, cargo, PATH setup, updates, and uninstall.
